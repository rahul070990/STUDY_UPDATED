Architecture:

1. Selenium Client-- code is converted into JSON whenever you trigger the test.
2. Browser Driver-- This JSON reaches the browser driver & respective browser communicates with respective Browser Driver through http
3. Browser-- Browser sends the response back to browser driver & finally returns the result back to Selenium Client
